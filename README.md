# Py Netcat

## Ho to use?

Help mode:

```bigquery
python python netcat.py -h
```

### Client example

```bigquery
python python netcat.py -t 0.0.0.0 -p 5555 -l -c
```

###Server example

```bigquery
python python netcat.py -t 0.0.0.0 -p 5555
```
